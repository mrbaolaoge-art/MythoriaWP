# 双币经济模型

## 1. 双币模型概述：古老智慧的现代演绎

### 1.1 玛雅双币哲学

在古代玛雅文明中，存在着"金币"与"玉币"的双重货币体系——金币用于日常交易，玉币用于神圣仪式和财富储藏。Mythoria神话之地将这一古老智慧与现代DeFi技术完美融合，创造了革命性的FUSD & MYR双币经济模型。

这个模型的核心理念是"一币驱动进入，一币驱动分红"，构建平台内生正循环的经济结构，实现稳定性与增长性的完美平衡。FUSD作为稳定的入金货币，确保了系统的价值基准；MYR作为收益分红代币，承载了生态的增长价值。

### 1.2 双币协同机制

Mythoria的双币模型不是简单的代币分离，而是一个有机的经济生态系统。玩家使用FUSD进入游戏进行PVP挑战，无论胜利还是失败，都会获得相应的MYR奖励或铸造权。整个过程中，AI国库系统持续监控和调控，确保两种代币的价值稳定和协同发展。

```
FUSD（稳定币）→ 游戏入金 → PVP挑战 → 胜利/失败 → MYR（游戏币）→ 分红收益
     ↑                                                        ↓
     ←————————————— AI国库调控 ←—————————————————————————————————
```

## 2. FUSD稳定币：神域的黄金货币

FUSD是Mythoria生态系统的稳定币，总发行量为2亿枚，部署在BSC链上。作为平台唯一的入金代币，FUSD承担着价值锚定和系统稳定的重要职责。

在分配结构上，4000万FUSD用于Swap1与USDT组成主池，另外4000万FUSD用于Swap2与MYR组成副池，这两个流动性池的LP代币都将永久锁定。剩余的1.2亿FUSD作为AI国库储备，用于价格稳定和智能调控。

FUSD的核心功能体现在三个方面。首先，它是平台唯一的入金代币，所有PVP挑战的投入、游戏结算和奖池分配都以FUSD为基准，确保了经济系统的统一性和可控性。其次，FUSD通过AI国库系统维持与USDT的1:1价值锚定，采用100% LP标准流动性设计，智能调控机制杜绝价格剧烈波动。最后，FUSD作为连接传统金融与GameFi生态的桥梁，实现价值在不同场景间的无缝流转，为整个经济系统提供稳定的价值基准。

## 3. MYR游戏代币：神域的翡翠货币

MYR是Mythoria的游戏收益代币，总发行量为20万亿枚，同样部署在BSC链上。作为分红导向的代币，MYR采用通缩型经济模型，通过多种机制确保长期价值增长。

所有的20万亿MYR都与4000万FUSD在Pancake上组成流动性池并锁定，流通供应量通过游戏机制和销毁机制动态平衡。MYR承载着多重功能：持币者可以参与平台收益分红，PVP奖励的50%以MYR形式发放，节点守护者的手续费分红也100%为MYR。

MYR的通缩价值机制是其核心特色。系统每日自动销毁一定数量的MYR，同时将15%的失败资金用于回购并销毁MYR，Swap2池还有每日销毁2%的机制来稳定池子。对于失败的玩家，他们不会完全损失价值，而是获得等值的MYR铸造额度，在钻石矿享受100%的铸造加成，将失败转化为新的价值创造机会。

## 4. Swap1主池：FUSD/USDT稳定基石

Swap1主池是整个经济系统的稳定基石，由4000万FUSD和200万USDT组成初始流动性，LP代币永久锁定以确保流动性安全。这个池子的设计理念是创造一个封闭式的交易环境，所有交易都在官方DApp内完成，杜绝外部套利和恶意操作，确保价格稳定和用户资金安全。

Swap1池与AI国库系统深度协同，AI系统24小时不间断监控价格波动，一旦价格偏离目标区间就立即执行调控操作，通过算法实现价格的动态平衡。每笔交易收取10%的手续费，这些手续费进入AI国库系统，形成稳定的内循环与价值回归机制。

FUSD的价格稳定性不依赖运气或承诺，而是靠机制保障。通过双池结构和AI调控确保结构性锚定，每次交易都有对应的反向操作实现即时对冲，而且交易越多，流动性越强，锚定效果越好。

## 5. Swap2副池：FUSD/MYR价值转换

Swap2副池由4000万FUSD和20万亿MYR组成，LP代币同样永久锁定以确保生态稳定。这个池子是所有游戏激励行为的释放入口，承担着FUSD与MYR之间价值转换的重要职责，连接游戏行为与经济收益。

Swap2池的独特之处在于其反向流动性生成机制。每次奖励发放都为池子注入流动性，红利分配与生态成长深度绑定，形成“参与→奖励→流动性→稳定→参与”的正向循环。每笔交易收取10%手续费，100%分给不同等级的节点，同时MYR每日销毁2%机制确保通缩效应。

通过Swap2池，MYR的价值得到充分发现。交易行为反映真实的供需关系，市场化的价格发现机制让MYR与FUSD的汇率能够准确反映MYR的内在价值。

## 6. AI国库系统：Kukulkan的智慧核心

### 6.1 系统概述

AI国库系统是Mythoria经济模型的核心大脑，被誉为"GameFi领域首个真正的AI国库系统"。它通过四大核心机制解决了传统GameFi项目通胀失控与价格脱钩的根本问题。

### 6.2 四大核心机制

第一个机制是原子对冲协议。每次用户买卖FUSD时，国库都会同步执行反向操作，实现即时双边对冲，压缩价格波动，确保价格快速回归稳定区间。这就像一个永不疲倦的做市商，始终在维护价格稳定。

第二个机制是Smootherstep调控函数。系统使用非线性的smootherstep数学函数进行价格调节，避免暴涨暴跌，智能稳控波动区间。这种算法能够根据市场情况动态调整调控强度，让价格变化更加平滑自然，避免剧烈波动对用户体验的影响。

第三个机制是LP强化机制。国库对冲交易会自动注入10%手续费到FUSD/USDT主池，持续增强流动性。这形成了一个正向循环：交易越多，手续费越多，LP增强越明显，价格越稳定，又吸引更多交易。

第四个机制是AI决策模型。系统实时监控FUSD/USDT汇率变化，分析买卖盘深度和流动性状况，识别用户交易行为模式和趋势。基于这些数据，AI能够提前识别潜在的价格风险，在最佳时机执行最适当的调控操作，并通过历史数据不断优化决策模型。

```
价格波动检测 → 原子对冲协议 → 即时稳定
       ↓              ↑
Smootherstep调控 → LP强化机制 → 流动性增强
       ↓              ↑
   AI决策模型 ←——————————————— 数据反馈学习
```

Mythoria的双币模型构建了一个完整的内生价值循环系统：

```
玩家入金(FUSD) → PVP挑战 → 胜利/失败分流
        ↓                      ↓
    AI国库调控              MYR铸造/奖励
        ↓                      ↓
    价格稳定 ←—————————— 节点分红/销毁机制
        ↓                      ↓
```

### 7.2 价值创造机制

特别值得一提的是，即使在PVP中失败，玩家也不会归零。失败者可以获得MYR铸造权，特别是钻石矿还能享受100%的铸造加成。这意味着随MYR价格的上涨，失败者也能获得更多收益，实现了真正的价值保值。

从长期来看，Mythoria的价值增长潜力巨大。通缩机制持续销毁MYR供应量，生态扩张将带来更多游戏模块和价值创造，而AI系统的持续优化也会不断提升整个系统的运行效率。
