# PVP对战与分配机制

## 1. 神选仪式：十室挑战核心玩法

### 1.1 房间对战机制

Mythoria的核心玩法"神选仪式"采用创新的十室挑战模式，每局游戏包含十个神秘房间，玩家需要选择其中一个作为藏身之所。当所有房间满员后，系统将触发开奖机制，通过区块链哈希算法随机生成0-9的决胜数字，对应淘汰一个房间，其余九个房间的玩家获得胜利。

这种设计确保了90%的高胜率，让更多玩家能够获得正向收益体验，同时10%的失败概率维持了游戏的经济平衡。被淘汰的玩家并非完全损失，而是进入"钻石矿"获得100%的MYR铸造权，将失败转化为新的价值创造机会。

### 1.2 场次分级体系

为满足不同投资能力和风险偏好的玩家需求，Mythoria设计了六个不同级别的挑战场次：

```
体验场 (50 FUSD)  → 新手友好，2分钟开奖
训练场 (100 FUSD) → 技能提升，2分钟开奖  
新手场 (500 FUSD) → 进阶挑战，2分钟开奖
学徒场 (1,000 FUSD) → 专业级别，2分钟开奖
战士场 (5,000 FUSD) → 高级玩家，2分钟开奖
精英场 (10,000 FUSD) → 顶级挑战，2分钟开奖
```

所有场次均采用2分钟快速开奖机制，确保游戏节奏紧凑刺激。投入金额从50 FUSD到10,000 FUSD的梯度设计，让不同经济实力的玩家都能找到适合的参与方式。

## 2. 收益分配机制：胜负皆有所得

### 2.1 胜利者奖励体系

获胜玩家将获得固定3%的红利收益，这个比例经过精密的经济模型计算，既能提供足够的激励，又能维持系统的可持续性。奖励以50% FUSD + 50% MYR的形式发放，确保玩家既能获得稳定的价值回报，又能参与MYR代币的长期增值。

奖励释放采用线性解锁机制，标准释放周期为12小时，但玩家可以通过提升星级等级来缩短释放时间，最快可缩短至2小时。这种设计鼓励玩家长期参与生态建设，通过星级成长获得更好的流动性体验。

### 2.2 失败者价值转化

失败玩家的投入资金不会简单消失，而是通过精心设计的分配机制实现价值的多重转化和循环利用：

**PVP挑战手续费分配结构**：
- **27%** - 玩家奖励：直接奖励参与游戏的玩家，作为获胜者的收益来源
- **50%** - 团队奖励：支持项目持续发展和技术迭代
- **15%** - 销毁机制：回购并销毁MYR代币，维持代币稀缺性
- **3%** - 全球节点：全球治理节点分红
- **5%** - 税收：合规运营资金

### 2.3 MYR铸造权机制

失败玩家获得等值的MYR铸造额度，这是Mythoria独创的"失败价值转化"机制。特别是在钻石矿中，失败者享受100%的铸造加成，意味着投入1000 FUSD失败后，可以获得价值1000 FUSD的MYR铸造权。

随着MYR代币价格的上涨，这些铸造权的实际价值也会相应增长。例如，如果MYR价格上涨50%，原本的1500 FUSD铸造权实际价值将达到2250 FUSD，让失败者也能分享生态成长的红利。

### 3.2 团队收益计算逻辑

星级系统的收益计算基于团队深度贡献，不依赖平台补贴或中心化分配。以不同星级的收益潜力为例：

**收益计算示例**（按不同MYR汇率）：

- **1星用户**：团队流水10万FUSD，获得0.45%分润 = 450 FUSD
  - MYR汇率0.1时：实际收益45 USDT
  - MYR汇率0.2时：实际收益90 USDT

- **5星用户**：团队流水200万FUSD，获得2.25%分润 = 45,000 FUSD

  - MYR汇率0.2时：实际收益9,000 USDT

- **10星用户**：团队流水8000万FUSD，获得4.50%分润 = 3,600,000 FUSD
  - MYR汇率0.1时：实际收益360,000 USDT
  - MYR汇率0.2时：实际收益720,000 USDT

这种设计确保了每一次星级提升都是结构性分润能力的跃迁，而非简单的头衔变化。

Mythoria推出V1-V7七个等级的节点NFT，总计1700张，作为生态治理和收益分配的核心载体。节点持有者享有Swap2交易手续费的分红权益，形成稳定的被动收入来源。

### 4.2 分红分配机制

节点分红来源于Swap2池100%的交易手续费，采用加权平均分配方式分给不同等级的节点。高等级节点用户不仅可以分配当前等级的手续费分红，还可以参与所有低等级节点池的分红，实现收益最大化。

**分红计算公式**：
```
节点分红 = (该等级用户数 + 加权用户数) / 当前池子手续费MYR总量
```

分红以MYR代币形式发放，每日UTC+0时间统一更新分配。这种设计体现了"节点等级越高，人数越少，分红越集中"的稀缺性价值，越早参与的用户获得越强的收益优势。

## 5. 释放机制与流动性管理

### 5.1 团队奖励释放规则

团队奖励采用双代币形式发放，用户领取时触发自动兑换机制。以1000 FUSD团队奖励为例，最终用户将获得450 FUSD等值的MYR + 405 FUSD，确保收益的多样化和风险分散。

### 5.2 MYR动态释放机制

MYR代币释放采用创新的动态调整模式，基础释放为每日1%的线性释放，但用户可以通过游戏活跃度获得加速释放额度：

**加速释放公式**：
```
每日加速释放额度 = 当日游戏流水 × 10%
```

这种设计鼓励用户保持游戏活跃度，通过参与游戏获得更好的资金流动性，形成"参与越多，释放越快"的正向激励循环。

### 5.3 节点MYR奖励

节点MYR奖励享有特殊待遇，采用100% MYR发放，即领即释，无需等待释放周期。这体现了节点持有者的特殊地位和权益，为节点NFT提供了额外的流动性价值。

## 6. 风险控制与平衡机制

### 6.1 经济模型平衡点

Mythoria的PVP机制经过精密的数学建模，确保长期经济平衡：

- **90%胜率**保证大多数玩家获得正向体验
- **3%固定收益**提供稳定的投资回报预期  
- **多层分配机制**确保资金在生态内有效循环
- **通缩销毁机制**维持MYR代币的长期价值增长

### 6.2 反作弊与公平性保障

系统采用区块链哈希算法生成随机数，确保开奖结果的绝对公平和不可操控。所有游戏记录和资金流向都在链上透明可查，杜绝任何形式的内幕操作或人为干预。

### 6.3 流动性风险管控

通过分级场次设计和星级成长体系，系统能够有效分散单一大额投入的风险，同时通过节点分红和团队奖励机制，确保各个层级的用户都能获得相应的收益保障。
